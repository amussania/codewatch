File opened directly — API calls will be blocked
Your browser blocks API calls when you open an HTML file directly from your desktop (file://). You need to serve it through a local server.
Fix — pick one of these options:
Option A (easiest): If you have Python, open Terminal, go to the folder with the file and run:
python3 -m http.server 8080
Then open http://localhost:8080/codewatch.html in your browser.

Option B: Use VS Code with the Live Server extension — right-click the file and click "Open with Live Server".

Option C: Upload the file to any static host (Netlify Drop, GitHub Pages) and open the URL.
